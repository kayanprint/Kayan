## 🚀 طريقة التشغيل السريع

### 1️⃣ **تحميل الملفات:**
```bash
# تحميل المشروع
git clone https://github.com/kayanprint/Kayan.git
cd Kayan/kayan_library_web/
```

### 2️⃣ **تثبيت المتطلبات:**
```bash
# تثبيت Flask
pip install flask

# أو تثبيت جميع المتطلبات
pip install -r requirements.txt
```

### 3️⃣ **تشغيل التطبيق:**
```bash
# تشغيل الخادم
python app.py

# سيعمل التطبيق على:
# http://localhost:5000
```

### 4️⃣ **الوصول من الهاتف:**
```bash
# إذا كان الهاتف على نفس الشبكة:
# http://[عنوان-IP-للكمبيوتر]:5000

# مثال:
# http://192.168.1.100:5000
```

## 📱 الميزات المحمولة الجديدة

### 🎨 **التصميم المتجاوب:**
- **شريط تنقل ثابت** في الأعلى مع أزرار مختصرة
- **شريط تنقل سفلي** للهواتف مع الوظائف الأساسية

- **قابل للتثبيت** على الهاتف كتطبيق عادي
- **يعمل أوفلاين** مع حفظ البيانات محلياً
- **تحديثات تلقائية** في الخلفية
- **إشعارات** (قريباً)
- **مشاركة سريعة** للفواتير

### 🎯 **الوظائف المحسنة:**
- **لوحة معلومات تفاعلية** مع إحصائيات مباشرة
- **تنبيهات المخزون المنخفض** بألوان تحذيرية
- **بحث سريع** في المنتجات والعملاء
- **إنشاء فواتير** محسن للمس
- **طباعة** محسنة للهواتف

## ⚙️ الإعدادات المتقدمة

### 🔧 **تخصيص الخادم:**
```python
# في ملف app.py
if __name__ == '__main__':
    init_database()
    app.run(
        debug=False,        # للإنتاج
        host='0.0.0.0',     # للوصول من الشبكة
        port=5000,          # المنفذ
        threaded=True       # دعم متعدد المستخدمين
    )
```

### 🌐 **إعداد الشبكة:**
```bash
# للوصول من الشبكة المحلية
python app.py

# للوصول من الإنترنت (يحتاج إعداد إضافي)
# استخدم خدمات مثل ngrok أو Heroku
```

### 📱 **تثبيت كـ PWA:**
1. افتح الموقع في المتصفح
2. اضغط على قائمة المتصفح (⋮)
3. اختر "إضافة إلى الشاشة الرئيسية"
4. سيظهر التطبيق كأيقونة على الهاتف

## 🛠️ استكشاف الأخطاء

### ❌ **مشاكل شائعة:**

#### **1. لا يمكن الوصول من الهاتف:**
```bash
# تأكد من أن الخادم يعمل على 0.0.0.0
python app.py

# تحقق من عنوان IP للكمبيوتر
ipconfig  # Windows
ifconfig  # Linux/Mac

# تأكد من أن الجدار الناري لا يحجب المنفذ 5000
```

#### **2. التطبيق لا يعمل أوفلاين:**
```javascript
// تحقق من تسجيل Service Worker
if ('serviceWorker' in navigator) {
    navigator.serviceWorker.register('/sw.js');
}

// تحقق من الكاش في أدوات المطور
// Application > Storage > Cache Storage
```

#### **3. الخطوط العربية لا تظهر:**
```css
/* تأكد من تحميل خط Cairo */
@import url('https://fonts.googleapis.com/css2?family=Cairo:wght@300;400;600;700&display=swap');

/* أو استخدم خط احتياطي */
font-family: 'Cairo', 'Arial Unicode MS', Arial, sans-serif;
```

#### **4. البيانات لا تحفظ:**
```python
# تحقق من أذونات ملف قاعدة البيانات
chmod 666 kayan_library.db

# تحقق من مسار قاعدة البيانات
DATABASE = 'kayan_library.db'
```
