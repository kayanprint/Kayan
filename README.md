## 🚀 التثبيت السريع

### 1. رفع القالب
1. انتقل إلى لوحة تحكم بلوجر
2. اذهب إلى **المظهر** > **تعديل HTML**
3. احذف الكود الموجود بالكامل
4. انسخ محتوى ملف `blogger-template.xml`
5. الصق الكود في المحرر
6. اضغط **حفظ**

### 2. إعداد AdSense
استبدل `ca-pub-XXXXXXXXXX` و `XXXXXXXXXX` بمعرف الناشر ومعرفات الوحدات الإعلانية الخاصة بك

### 3. تخصيص المعلومات
في ملف القالب، ابحث عن `STORE_CONFIG` وقم بتحديث المعلومات
