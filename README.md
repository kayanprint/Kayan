## 📖 نظرة عامة

قالب **مكتبة ومطبعة كيان** هو حل متكامل لتحويل مدونة بلوجر إلى متجر إلكتروني احترافي مع جميع الميزات المطلوبة للتجارة الإلكترونية في مصر.

### ✨ الميزات الرئيسية

- 🛍️ **تحويل المقالات إلى منتجات** - تلقائياً
- 🛒 **عربة تسوق متطورة** - مع localStorage
- 💳 **بوابات دفع مصرية** - فودافون، أورانج، اتصالات، فوري
- 📦 **نظام شحن متكامل** - جميع محافظات مصر (27 محافظة)
- 📊 **ربط Google Sheets** - حفظ الطلبات تلقائياً
- 📱 **تكامل WhatsApp** - إرسال الطلبات والفواتير
- 🔐 **لوحة إدارة آمنة** - متابعة الطلبات والمبيعات
- 💰 **تحسين AdSense** - 10+ موقع إعلاني محسن
- 📱 **تصميم متجاوب** - يعمل على جميع الأجهزة
- 🎨 **تصميم عربي أنيق** - مخصص للسوق المصري

## 🚀 البدء السريع

### 1. التحميل والتثبيت
```bash
# حمل الملفات
git clone https://github.com/kayanprint/kayan-blogger-store.git

# أو حمل ملف ZIP مباشرة
```

### 2. رفع القالب
1. اذهب إلى بلوجر → المظهر → تعديل HTML
2. ارفع ملف `kayan-template-basic.xml`
3. احفظ التغييرات

### 3. التخصيص الأساسي
- غير معلومات الاتصال
- حدث بيانات الدفع
- اربط Google Sheets
- فعل AdSense

📚 **[دليل التثبيت المفصل](docs/installation-guide.md)**

## 💡 كيف يعمل النظام

### 1. تحويل المقالات إلى منتجات
```javascript
// يتم استخراج السعر تلقائياً من المحتوى
var priceMatch = content.match(/(\d+)\s*جنيه/);
var price = priceMatch ? priceMatch[1] : '0';
```

### 2. نظام عربة التسوق
- حفظ محلي باستخدام `localStorage`
- تحديث تلقائي للكميات والأسعار
- حساب الشحن حسب المحافظة

### 3. معالجة الطلبات
1. **العميل** يملأ نموذج الطلب
2. **النظام** يحفظ في Google Sheets
3. **WhatsApp** يفتح تلقائياً مع تفاصيل الطلب
4. **الإدارة** تتابع من لوحة التحكم

## 📱 إضافة المنتجات

### طريقة سهلة وسريعة
1. **أنشئ مقال جديد** في بلوجر
2. **اكتب عنوان المنتج** كعنوان للمقال
3. **أضف صورة بارزة** للمنتج
4. **اكتب الوصف والسعر**:
   ```
   وصف المنتج المفصل هنا...
   
   السعر: 150 جنيه
   
   مميزات إضافية...
   ```
5. **أضف التصنيفات** (طباعة، مكتبة، تصميم)
6. **انشر المقال** ✅

### نصائح للمنتجات الناجحة
- 📸 **صور عالية الجودة** - واضحة وجذابة
- 📝 **وصف مفصل** - يشرح الفوائد والمميزات
- 🏷️ **تصنيفات دقيقة** - لسهولة البحث
- 💰 **أسعار واضحة** - بصيغة "X جنيه"

## 🔧 استكشاف الأخطاء

### مشاكل شائعة

#### ❌ لا تظهر الأسعار
**الحل**: تأكد من كتابة السعر بصيغة "150 جنيه"

#### ❌ لا تعمل عربة التسوق
**الحل**: تحقق من تحميل ملفات JavaScript

#### ❌ لا تحفظ الطلبات في Google Sheets
**الحل**: راجع إعدادات Google Apps Script

#### ❌ لا تظهر الإعلانات
**الحل**: تأكد من رقم الناشر الصحيح في AdSense

📚 **[دليل استكشاف الأخطاء الكامل](docs/troubleshooting.md)**
