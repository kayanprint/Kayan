## 📋 نظرة عامة

نظام إدارة متكامل لمكتبة ومطبعة كيان يوفر حلول شاملة لإدارة المنتجات والعملاء والموردين والفواتير مع إمكانيات طباعة متقدمة ودعم كامل للغة العربية.

### ✨ المميزات الرئيسية

- 📦 **إدارة شاملة للمنتجات والمخزون**
- 👥 **إدارة العملاء والموردين**
- 🧾 **نظام فواتير متكامل مع حساب تلقائي**
- 📊 **تقارير مفصلة قابلة للطباعة**
- 🖨️ **طباعة بأحجام A4 و B5**
- 🔄 **تحديث المخزون التلقائي**
- 🌐 **دعم كامل للغة العربية**
- 💾 **نسخ احتياطي تلقائي**

---

## 🎯 الحلول المتاحة

تم إنشاء **4 حلول مختلفة** لتناسب احتياجاتك التقنية:

### 1️⃣ نظام ويب تفاعلي (Flask + SQLite)
📁 **المجلد:** `kayan_library_web/`

**المميزات:**
- واجهة ويب حديثة ومتجاوبة
- قاعدة بيانات SQLite خفيفة
- دعم متعدد المستخدمين
- إمكانية الوصول من أي جهاز

**طريقة التشغيل:**
```bash
cd kayan_library_web
pip install flask
python app.py
# افتح http://localhost:5000
```

### 2️⃣ سكريبت SQL لـ Microsoft Access
📁 **المجلد:** `kayan_library_access/`

**المميزات:**
- ملف SQL جاهز للاستيراد
- دليل مفصل لإنشاء النماذج والتقارير
- أكواد VBA للوظائف التلقائية

**طريقة الاستخدام:**
1. افتح Microsoft Access
2. استورد ملف `database_structure.sql`
3. اتبع دليل `forms_guide.md`

### 3️⃣ تطبيق Python سطح المكتب
📁 **المجلد:** `kayan_library_python/`

**طريقة التشغيل:**
```bash
cd kayan_library_python
pip install -r requirements.txt
python main.py
```

### 4️⃣ دليل إنشاء مفصل لـ Access
📁 **المجلد:** `kayan_library_guide/`

دليل شامل خطوة بخطوة لإنشاء النظام في Microsoft Access من الصفر.
